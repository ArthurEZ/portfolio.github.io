My Motivation
แรงบันดาลใจ


สิ่งที่ผมถนัดที่สุดคือการนำความรู้ที่มีนำมาประยุกต์ใช้ในชีวิตจริงและผมก็ชอบที่ได้สร้างหรือซ่อมแซมสิ่งต่างๆด้วยตัวเอง ทุกครั้งที่ทำสำเร็จมันทำให้ผมรู้สึกดีและภูมิใจในสิ่งที่ทำได้ หรือแม้แต่หลังจากที่เข้าเรียนสายวิทยาศาสตร์-คณิตศาสตร์ผมก็นำความรู้ทางวิทยาศาสตร์มาใช้ในชีวิตจริงเสมอ หรือการนำกระบวนการทางวิทยาศาสตร์มาหาคำตอบในคำถามที่ผมสงสัยขึ้นมา ในตอนเป็นเด็กผมชอบเล่นเกมเป็นอย่างมาก เกมที่ผมชอบเล่นที่สุดคือ Minecraft เพราะว่าเกมนี้ทำให้ผมได้ปลดปล่อยจินตนาการในวัยเด็กอย่างเต็มที่ ซึ่งหลังจากนั้นมันเริ่มทำให้ผมรู้สึกอยากทำเกมและสนใจในคอมพิวเตอร์ขึ้นมาแต่ในตอนนั้นไม่สามารถทำได้เนื่องจาก ผมยังไม่มีความรู้ความสามารถมากพอและเนื่องด้วยมัธยมต้นผมนั้นเรียนหนักมากเพื่อที่จะได้เรียนต่อในที่ๆดี ผมก็เก็บความฝันนี้ไว้

ในช่วงมัธยม 5 ผมตื่นเต้นมากที่จะได้เรียนเขียนโค้ดพื้นฐานเพราะเป็นสิ่งที่ตนเองสนใจมาตั้งแต่เด็ก ภาษาที่ได้เรียนคือภาษา C++ แต่ในตอนนั้นผมยอมรับว่าผมตามอาจารย์ไม่ทันเลยจึงทำให้ผมท้อแท้กับการเรียนสายคอมพิวเตอร์และได้หยุดความคิดและความฝันไว้ ในช่วงขึ้นมัธยม 6 ผมถูกคนรอบตัวถามว่าได้คณะที่สนใจรึยังอยู่เสมอในตอนแรกผมนั้นก็ยังไม่รู้ตัวเอง ผมเลยถามตัวเองว่าสิ่งที่ผมถนัดนั้นคืออะไร และชอบอะไร ผมนั้นถนัดในวิชาฟิสิกส์ คณิตศาสตร์ และผมชอบในดนตรีและเกม จึงทำให้ผมหันกลับมาลองเริ่มต้นกับการ coding อีกครั้ง ครั้งนี้ผมเรียนรู้ด้วยตัวเองและเริ่มที่ภาษา python และครั้งนี้ผมกับรู้สึกว่ามันก็ไม่ได้ยากอย่างถึงขั้นทำไม่ได้เลยและผมก็ได้ลองศึกษา html, css, JavaScript, และ C# ทำให้ผมกลับมามีไฟอีกครั้งและผมเริ่มมีความสนใจในวิชาชีพ Software Engineer จึงทำให้ผมตัดสินใจที่จะเข้าคณะวิศวกรรมศาสตร์ และตั้งใจเรียนเพื่อให้ได้เข้าสาขาวิศวกรรมคอมพิวเตอร์

BunTong! คือเกมแนว Vocabulary Quiz โดยมีไก่ที่ชื่อบุญโต้งต้องปราบปีศาจโดยผู้เล่นจะต้องทายความหมายของคำศัพท์ให้ถูกต้องเพื่อที่จะทำให้บุญโต้งโจมตีได้ แต่หากผู้เล่นทายผิดบุญโต้งจะโดนปีศาจโจมตี ผู้เล่นมีโอกาสผิดทั้งหมด 3 ครั้ง ก่อนที่บุญโต้งจะรับไม่ไหวและหมดสติไป

เมื่อตอนเป็นเด็กผมมีความฝันที่อยากที่จะมีเกมเป็นของตัวเอง แต่ด้วยตอนนั้นผมยังไม่มีความรู้และความสามารถมากพอที่จะทำ ผมก็เลยสานฝันวัยเด็กตัวเองด้วยการลองทำเกมขึ้นมา Engine ที่ผมใช้คือ Unity การทำเกมครั้งนี้ทำให้ผมรู้ว่ามันไม่ใช่เรื่องง่ายเลยที่จะสร้างเกมขึ้นมา ตั้งแต่การออกแบบจนไปถึงการวางระบบแต่เนื่องด้วยเวลาในการทำที่มีแค่สัปดาห์เดียวจึงทำให้บางอย่างที่วางไว้ต้องตัดออกเพื่อให้ทันเวลา
